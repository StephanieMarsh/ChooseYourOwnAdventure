How to Play the Choose Your Own Adventure Game (CYOA)

1. download CYOA.py
2. open terminal
3. route to the correct directory of where the game is
4. run the game by typing python3 CYOA.py
    note: it will import the random and time packages to play
5. play the game in terminal by making choices, entering w or s as necessary.

Tip: there is a lot of text to read, so it is recommended to make your terminal window wider than it usually is.
